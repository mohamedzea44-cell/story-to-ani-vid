# AnimeCast — منصة تحويل قصص الأنمي إلى حلقات فيديو

## الفكرة
تكتب قصة، والذكاء الاصطناعي يحوّلها لـ"حلقة أنمي" مدتها تبدأ من 10 دقائق وقابلة للزيادة، مع صور مشاهد أنمي عصرية، تعليق صوتي، أصوات شخصيات، موسيقى ومؤثرات. تنشئ عدة حلقات، تعدّلها، وتشاركها برابط عام.

## ملاحظة مهمة حول "الفيديو"
تصدير ملف MP4 حقيقي مدته 10 دقائق غير ممكن على الباك (Cloudflare Workers ما فيه Chromium/ffmpeg). الحل: **مشغّل حلقة داخل المتصفح** — يعرض المشاهد بالتزامن مع الصوت (تجربة مطابقة لمشاهدة فيديو، ورابط المشاركة يفتح المشغّل مباشرة مثل YouTube). لو أردت تنزيل MP4 لاحقًا نضيف تصدير عبر المتصفح كخطوة ثانية.

## ملاحظة حول التكلفة والوقت
حلقة 10 دقائق ≈ 40-60 مشهد. لكل مشهد: صورة AI + تعليق صوتي. توليد الحلقة الكاملة يستغرق عدة دقائق ويستهلك رصيد AI معتبر لكل حلقة. سنعرض شريط تقدم واضح ونحفظ كل شيء في قاعدة البيانات حتى لا يضيع العمل.

## نطاق الإصدار الأول

### الميزات
- **تسجيل دخول**: Google + إيميل عبر Lovable Cloud
- **لوحة "حلقاتي"**: قائمة الحلقات + إنشاء/تعديل/حذف/تكرار حلقة (بدون حد للعدد)
- **محرر الحلقة**:
  - عنوان الحلقة + رقم الحلقة (S1E1, S1E2...)
  - كتابة القصة (عربي/إنجليزي)
  - اختيار **المدة المستهدفة**: 10 / 15 / 20 / 30 دقيقة (سلايدر، الافتراضي 10)
  - اختيار الاستايل البصري: شونين حديث / شوجو / سايبربانك / غيبلي / أنمي داكن
  - اختيار المزاج العام: أكشن / درامي / رومانسي / غموض / كوميدي
  - زر **"قسّم القصة لمشاهد"** — Gemini يمدّ القصة إن كانت قصيرة ليملأ المدة، ويقسّمها لمشاهد (كل مشهد: وصف بصري، سرد الراوي، حوار، شخصية متكلمة، مزاج، مدة مقترحة بالثواني)
  - جدول مشاهد قابل للتعديل بالكامل: تعديل النص، تغيير الشخصية، تعديل مدة المشهد بدقة، إضافة/حذف/إعادة ترتيب المشاهد، إعادة توليد صورة أو صوت لأي مشهد
  - **إدارة الشخصيات**: قائمة شخصيات الحلقة، لكل شخصية اسم + وصف مظهر (يستخدم في توليد الصور) + صوت مختار (voice)
  - أزرار توليد: "ولّد كل الصور" / "ولّد كل الأصوات" مع شريط تقدم لكل مشهد
- **الاستوديو (معاينة)**: تشغيل الحلقة كاملة مع مزامنة الصور + الراوي + الحوار + الموسيقى + الترجمة
- **صفحة المشاهدة العامة `/watch/$slug`**: تفتح لأي شخص، فيها المشغّل + عنوان الحلقة + الحلقات الأخرى للسلسلة
- **مشاركة**: نسخ رابط، معاينة Open Graph (صورة الغلاف + العنوان) لروابط WhatsApp/تويتر
- كل زر وعنصر في الموقع يفتح وجهة حقيقية — لا شيء وهمي

### خارج نطاق الإصدار الأول
- تنزيل ملف MP4
- رسوم متحركة كاملة (frame-by-frame)
- تعليقات/إعجابات من المشاهدين
- اشتراكات مدفوعة

## المكدس التقني (للمرجع)
- Frontend: TanStack Start + Tailwind + shadcn + Motion
- Backend: Lovable Cloud (Postgres + Auth + Storage)
- AI: Gemini للتقسيم والتوسيع، Gemini Image لصور الأنمي، `openai/gpt-4o-mini-tts` للأصوات (أصوات مختلفة لكل شخصية)
- موسيقى/SFX: مكتبة جاهزة CC0 نرفعها للـ Storage (6-8 مقاطع لكل مزاج)
- المشغّل: React + Motion + عنصر `<audio>` مسبق التحميل لكل مشهد، متزامن بالوقت

## قاعدة البيانات (مع RLS و GRANTs)
```
episodes(id, user_id, series_title, episode_number, title, story_text,
         language, style, mood, target_duration_min, status,
         share_slug UNIQUE, cover_image_url, total_duration_sec, created_at)
scenes(id, episode_id, order_index, description, narration, dialogue,
       character_id, duration_sec, image_url, narration_audio_url,
       dialogue_audio_url, sfx_url, image_status, audio_status)
characters(id, episode_id, name, appearance_prompt, voice)
```
- RLS: المالك يقرأ/يعدّل حلقاته فقط
- سياسة `TO anon SELECT` على `episodes` عبر `share_slug` وعلى مشاهدها للعرض العام
- Storage bucket عام `episode-assets` للصور والأصوات

## الصفحات (كلها routes حقيقية)
```
/                       Landing (شرح + CTA)
/auth                   دخول/تسجيل
/_authenticated/
  dashboard             حلقاتي (كل الحلقات + مجمّعة بالسلسلة)
  episodes/new          إنشاء حلقة جديدة
  episodes/$id          محرر (قصة + مشاهد + شخصيات + توليد)
  episodes/$id/preview  استوديو المعاينة
/watch/$slug            مشغّل عام
```

## خطوات التنفيذ
1. تفعيل Lovable Cloud + Auth (Email + Google) + Storage
2. الهجرات: جداول + RLS + GRANTs + trigger profile
3. Landing + Auth + Dashboard
4. صفحة إنشاء حلقة + محرر القصة والإعدادات
5. Server functions للـ AI: `expandAndSplitStory`, `generateSceneImage` (streaming), `generateSceneAudio` (SSE TTS)، مع حفظ كل نتيجة في DB
6. جدول المشاهد التفاعلي + إدارة الشخصيات
7. المشغّل (معاينة داخلية + صفحة عامة)
8. زر مشاركة + Open Graph tags على `/watch/$slug`
